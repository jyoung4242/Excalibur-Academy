# Welcome to TypeScript

You're about to learn one of the most powerful tools in modern software development — and you're going to do it by building games.

## What is TypeScript?

TypeScript is a **typed superset of JavaScript**. That means every JavaScript program is already valid TypeScript. TypeScript just adds a layer on top that helps you write better, more reliable code.

Think of it like this:

> JavaScript is a sports car with no dashboard gauges. TypeScript gives you the speedometer, fuel gauge, and warning lights.

## Why Learn TypeScript Through Games?

Games are the perfect learning environment for TypeScript because they demand:

- **Clear data structures** — a player, an enemy, a position all have specific shapes
- **Typed functions** — move(), attack(), and respawn() take specific inputs and return specific outputs
- **Refactoring confidence** — change a property name once and TypeScript tells you everywhere to update

By the time you finish this track, you'll have learned TypeScript concepts that apply to *any* software project — and you'll have built games along the way.

## What You'll Need

You don't need to install anything to get started. All the interactive examples in this course run directly in your browser using the **Excalibur Playground**.

For real projects, you'll want:

- [Node.js](https://nodejs.org) (v18 or newer)
- [VS Code](https://code.visualstudio.com) with the TypeScript extension

## How Lessons Work

Each lesson follows this pattern:

1. **Concept** — Learn a TypeScript idea with clear examples
2. **Game Context** — See how it applies in a game scenario
3. **Playground** — Try it live in the browser
4. **Challenge** — A small exercise to solidify understanding
